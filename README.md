# pyinfraformat
Python library for Finnish Infraformat (version 2.3)

## Installation

With pip

    pip install git+https://github.com/ahartikainen/pyinfraformat

Or `git clone`

    git clone https://github.com/ahartikainen/pyinfraformat
    cd pyinfraformat
    pip install .
